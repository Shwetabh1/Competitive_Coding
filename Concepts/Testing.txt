##TESTING##

1.Smoke testing -- initial builds the software is tested if it starts and basic operations is performed or not
2.Sanity testing  -- working software it checks if the bug fixes etc is all good or not
3.Regression Testing/System Testing ---after smoke testing it checks if everything is fine
4.Unit testing(Mocha, Chai etc in Node.JS | Jasmine, Protractor in Angular)
5.End-to-End testing
6.Integration testing
7.Automation Testing
Selenium mocks the browser, Automates and checks.
